介绍

  ABACUS（原子算筹）作为一款高效、开源的第一性原理材料计算软件，不仅支持 CPU，还兼容 GPU、DCU 等多种异构硬件设备，其平面波基组模块能够在不同计算平台上处理不同类型的材料体系，提供灵活可靠的计算支持。为了验证该软件在不同硬件平台上的正确性和效率，开发团队提供了一系列算例和结果供对比，为跨平台计算结果的一致性提供参考依据。本算例集选取10个典型测试案例，涵盖半导体、分子、二维材料、金属表面、钙钛矿、金属单质、铁磁材料、液态体系、电池材料等多种代表性体系。

软件和算例准备

以上 10 个算例可在 gitee 或者 GitHub 上免费下载。具体方式是：在网页右侧点击 克隆/下载-> 下载 ZIP 得到算例压缩包并解压，或在命令行通过如下命令下载算例：

  1. gitee 网站
git clone https://gitee.com/mcresearch/abacus-user-guide.git
  1. GitHub 网站
git clone https://github.com/MCresearch/abacus-user-guide.git

进入目录 /abacus-user-guide/examples/ ，pw2-cg 和 pw2-ds 子目录下有十个算例，分别对应本算例集 ks_solver=cg 和 ks_solver=dav_subspace，并且配备了相应的赝势文件，无需额外下载。

参数设置

手动设置 fft 格子为 ≥ 自动生成大小的 2 的幂次；对 010_216Si 算例，降低了 nbands 和 ecutwfc。

分别测试 ks_solver = cg/dav_subspace。

使用 pseudo-dojo v0.5 赝势（NC SR ONCVPSP v0.5，PBE，standard accuracy，upf）。
