ABACUS 平面波基组下的 10 个典型算例

ABACUS 平面波基组下的 10 个典型算例

作者：陈诺，邮箱：cn037@stu.pku.edu.cn

审核：陈默涵，邮箱：mohanchen@pku.edu.cn

最后更新时间：2025/12/15

介绍

ABACUS（原子算筹）作为一款高效、开源的第一性原理材料计算软件，不仅支持 CPU，还兼容 GPU、DCU 等多种异构硬件设备，其平面波基组模块能够在不同计算平台上处理不同类型的材料体系，提供灵活可靠的计算支持。为了验证该软件在不同硬件平台上的正确性和效率，开发团队提供了一系列算例和结果供对比，为跨平台计算结果的一致性提供参考依据。本算例集选取10个典型测试案例，涵盖半导体、分子、二维材料、金属表面、钙钛矿、金属单质、铁磁材料、液态体系、电池材料等多种代表性体系。

软件和算例准备

以上10个算例可在gitee或者GitHub上免费下载。具体方式是：在网页右侧点击克隆/下载-> 下载 ZIP得到算例压缩包并解压，或在命令行通过如下命令下载算例：

  1. gitee 网站
git clone https://gitee.com/mcresearch/abacus-user-guide.git
  1. GitHub 网站
git clone https://github.com/MCresearch/abacus-user-guide.git

进入目录 /abacus-user-guide/examples/pw 文件夹，下有十个算例，并且配备了相应的赝势文件，无需额外下载。
